## 🛠 習得・活用テクニック
今回の制作で特に意識した、または新しく習得した技術のまとめです。

### 1. 画像制御のマスター
- `object-fit: cover` を使用し、比率を維持したまま画像を枠いっぱいに表示。
- `100vh` を用いたメインビジュアルの全画面表示。
- `object-position: center top` による、レスポンシブ時の最適なトリミング位置の指定。

### 2. レイアウトの最適化
- `calc(100% / 3)` を活用し、割り切れない3等分レイアウトを正確に実装。
- `display: flex` と `gap` プロパティによる、現代的でクリーンな余白管理（`last-child` に頼らない設計）。
- 親要素（`#footer`）への `font-size` 継承による、DRY（Don't Repeat Yourself）なコード記述。

### 3. レスポンシブ設計の「守り」
- `padding: 0 20px` を通常時に設定することで、ブレイクポイント付近での文字の「端くっつき」を未然に防ぐ設計。
